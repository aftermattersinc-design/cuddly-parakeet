Connect your Stripe account

After Matters uses Stripe Payment Links - the simplest way to take real payments with no code. Once you've created your links in Stripe, paste them below and they'll be saved for this device. When a customer clicks "Pay securely with Stripe" they'll go straight to your Stripe checkout page.

1
Create a free Stripe account at stripe.com/nz. Complete identity verification - Stripe requires this before you can receive payments.
2
In your Stripe dashboard, go to Payment Links → Create payment link. Create one link for each product: Vault Setup ($249), Family Vault ($399), Bereavement Pass ($199). Set currency to NZD.
3
For each link, set the success redirect URL to https://aftermatters.co.nz/onboarding.html so customers land on your onboarding flow after payment.
4
Copy each Payment Link URL (looks like buy.stripe.com/xxxx) and paste it into the fields below. Click Save - done.
5
For the free Shoreline plan - no Stripe needed. The button links directly to signin.html to create a free account.
